DIGITAL GRASS

DOWNLOAD INSTRUCTIONS

  -Digital Grass is a text-based game centering around, as implied, digital grass!

  -To play the game, simply download the .bas file and run it in any  QBASIC emulator.

  -If you wish to play without downloading emulators, simply copy and paste the entire contents of the .txt file into QBJS.org, and press run.

In Digital Grass you can...

  -Purchase items in the shop!
  -Water and fertilize your grass!
  -Enjoy humorous daily musings!
  -Or, (if you are dedicated enough), FIND THE ELUSIVE SECRET END SCREEN!!!
